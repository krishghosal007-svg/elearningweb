🚀 Learning Platform Web App

A modern and responsive learning platform UI built using HTML, CSS, and JavaScript. This project provides an interactive course browsing experience with filtering, search functionality, and a student dashboard.

🌟 Features

✨ Course Filtering System

Filter courses by category dynamically
Smooth UI transitions and animations

🔍 Search Functionality

Real-time course search
Instant filtering based on user input

📚 Course Details Page

Structured course modules
Expandable content layout

📊 Dashboard Analytics

Learning activity visualization using Chart.js
Weekly progress tracking


⚙️ Functionality Overview
🔹 Course Filtering
Users can filter courses by category
Active filter button UI updates dynamically
🔹 Search Bar
Real-time filtering of courses by title
Case-insensitive matching
🔹 Dashboard Chart
Displays learning hours per week
Smooth gradient line chart using Chart.js
🔹 Completion System
Checkbox to mark lecture as completed
Visual UI feedback on completion
